Git

Основные команды Git

git init
Инициализирует новый репозиторий Git в текущей директории
git clone
Клонирует удаленный репозиторий на локальную машину
git add
Добавляет изменения в индекс для последующего коммита
git commit
Фиксирует изменения в репозитории
git push
Отправляет изменения в удаленный репозиторий

Полезные команды

  • git status - показывает состояние рабочей директории
  • git pull - получает изменения из удаленного репозитория
  • git branch - работает с ветками
  • git merge - объединяет ветки
